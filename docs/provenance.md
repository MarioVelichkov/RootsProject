# Provenance

The computer vision pipeline and trained U-Net model are based on Mario Velichkov's
original university project work.

The OT-2 simulation assets and parts of the PID integration were adapted from a
completed reference implementation supplied with the project materials. This includes
the URDF files, meshes, textures, work-envelope values, and coordinate-conversion
experiments.

The repository contains only the source code and assets required to run the current
pipeline. Raw coursework files, presentations, experiments, and duplicate working
copies are excluded.
