# Task Summary

The original project contained a broad set of plant computer vision and robotics tasks. The portfolio reconstruction keeps only the parts that support a polished end-to-end demo.

| Area | Original task focus | Portfolio treatment |
|---|---|---|
| Tasks 1-3 | Annotation, Petri dish crop, plant segmentation | Preserved as background and source material |
| Tasks 4-5 | Dataset prep and U-Net training/inference | Refactored into the active CV inference path |
| Tasks 6-8 | Root segmentation, RSA, root length/Kaggle pipeline | Refactored into root tip extraction and diagnostics |
| Tasks 9-10 | OT-2 simulation and PID control | Refactored into active simulation demo |
| Task 11 | Reinforcement learning controller | Archived only, not part of the active product |
| Tasks 12-13 | Integration and benchmarking | Rebuilt as PID-only integration plus simple metrics |
| Presentation | Academic presentation deliverable | Excluded |
